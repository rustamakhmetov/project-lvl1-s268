```
npm install -g ra_brains_games
```

### Игры

1. Определить четность
```
brain-even
```

2. Калькулятор
```
brain-calc
```

3. Наименьший общий делитель
```
brain-gcd
```

4. Сбалансировать число
```
brain-balance
```

5. Арифметическая прогрессия
```
brain-progression
```

5. Простое число
```
brain-prime
```
